# 소수박물관 소개 웹페이지

경상북도 영주에 있는 **소수박물관**(유교전문박물관)을 소개하는 정적 웹사이트입니다.

## 실행 방법

로컬에서 바로 열거나 간단한 서버로 확인할 수 있습니다.

```bash
# Python 내장 서버
python3 -m http.server 8080

# 또는 index.html을 브라우저에서 직접 열기
```

브라우저에서 `http://localhost:8080` 으로 접속합니다.

## 구성

- `index.html` — 메인 페이지
- `css/styles.css` — 스타일
- `js/main.js` — 모바일 메뉴 등 경량 스크립트
